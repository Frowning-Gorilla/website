Website privacy

What this site collects, and what it doesn't.

This website is run by Frowning Gorilla Ltd ("Frowning Gorilla", "we", "us"), a company registered in England and Wales.

The short version

This website sets no cookies, runs no analytics, and tracks nothing. We don't know who you are, how you got here, or what you looked at. There is no consent banner because there is nothing to consent to.

What the site loads

Every part of this site is served from our own domain. The fonts are hosted here rather than fetched from Google Fonts, so no third party sees your visit. No advertising, no trackers, no embedded scripts from other companies.

The site is hosted on GitHub Pages. Like any web host, GitHub processes the requests needed to serve pages to you. See GitHub's privacy statement.

If you get in touch

There are no forms on this site. Every contact link is an ordinary email link, so writing to us sends an email from your own email account, exactly as if you had typed the address yourself. Nothing passes through a third-party form service.

We use what you send only to answer you. We don't add you to a mailing list, and we don't pass it on to anyone else.

Changes

If this ever changes, we'll change this page. There is no mailing list to notify, because we don't have your address unless you've written to us.
